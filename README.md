# vue-playground
Common patterns and features used when developing Vue.JS applications.

## Setup
```
yarn install
```

## Scripts

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
